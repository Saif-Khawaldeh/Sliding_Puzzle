# Sliding Puzzle

This project is my first Visual Basic (VB) project.

It was built as a self-imposed programming challenge and prepared as part of my portfolio submission for entry into a prestigious university course.

The application is a classic sliding puzzle game where the objective is to reorder the boxes into their correct positions.

## Requirements

- Windows OS
- .NET 5.0 SDK (or Visual Studio with .NET desktop development tools)

## How to Run

### Option 1: Visual Studio (recommended)

1. Open `SlidingPuzzle.sln` in Visual Studio.
2. Restore/build the solution if prompted.
3. Run the app using `F5` (or the Start button).

### Option 2: Command Line

1. Open a terminal in the project root folder.
2. Run:
   - `dotnet restore`
   - `dotnet build SlidingPuzzle/SlidingPuzzle.vbproj`
   - `dotnet run --project SlidingPuzzle/SlidingPuzzle.vbproj`
